## OMZsh
1. Install zsh
```macos
brew install zsh 
```
> before install check ``` zsh --version ``` if you got 5.0.8 you can skip install section
2. Set zsh as default shell
> For macOS younger than High Sierra use:
```macos
chsh -s /usr/local/bin/zsh
```
> For macOs High Siera or older use:
```macos
chsh -s /bin/zsh
```
